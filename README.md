# 🚀 Ringta GUI — Универсальный хаб для Roblox

[![Version](https://img.shields.io/badge/version-2.0-purple)](https://github.com/Raphatlia/RingtaGUI)
[![License](https://img.shields.io/badge/license-MIT-blue)](https://github.com/Raphatlia/RingtaGUI)
[![Roblox](https://img.shields.io/badge/Roblox-99Nights-ff69b4)](https://www.roblox.com)

> **Премиум-интерфейс для любых игр Roblox.**  
> Стильный дизайн, аккордеоны, темы, стекло и macOS-кнопки.


---

## ✨ Особенности

### 🎨 **Дизайн**
- **MacOS-стиль** — красная, жёлтая, зелёная кнопки
- **Эффект стекла** — полупрозрачный фон
- **Плавные анимации** — при открытии и переключении
- **6 готовых тем** — фиолетовая, красная, синяя, зелёная, оранжевая

### 🧩 **Функционал**
- **4 вкладки**: Home, Combat, Farm, Settings
- **Аккордеоны** — скрываемые блоки с настройками
- **Переключатели** — вкл/выкл с анимацией
- **Профиль игрока** — аватар и никнейм
- **Перетаскивание** — иконки и меню

### ⚡ **Производительность**
- Лёгкий и быстрый
- Работает на любом эксплойте (Delta, Synapse, KRNL)
- Оптимизирован для мобильных устройств
