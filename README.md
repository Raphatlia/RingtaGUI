## ✨ Особенности

### 🎨 **Дизайн**
- **MacOS-стиль** — красная, жёлтая, зелёная кнопки
- **Эффект стекла** — полупрозрачный фон
- **Плавные анимации** — при открытии и переключении
- **6 готовых тем** — фиолетовая, красная, синяя, зелёная, оранжевая

### 🧩 **Функционал**
- **4 вкладки**: Home, Combat, Farm, Settings
- **Аккордеоны** — скрываемые блоки с настройками
- **Переключатели** — вкл/выкл с анимацией
- **Профиль игрока** — аватар и никнейм
- **Перетаскивание** — иконки и меню

### ⚡ **Производительность**
- Лёгкий и быстрый
- Работает на любом эксплойте (Delta, Synapse, KRNL)
- Оптимизирован для мобильных устройств
